# greek-mythology-trivia
This repository contains a fully functional Greek Mythology Trivia Web App made for a JavaScript final project. Built using HTML, CSS, and JavaScript. This project features a fully interactive quiz system where users answer randomized questions about Greek gods, heroes, and myths. Includes dynamic scoring and custom question creation.


Features will be:

- 15 Greek mythology questions (+ your own added questions!)
- Multiple choice format
- Score tracking
- Final score screen

- Custom question form where you can add your own question
- Questions are saved in memory


Technologies used:
- HTML
- CSS
- JavaScript

How to run:
1. Download or clone repository
2. Open index.html in your web browser
3. Click start game to begin
4. Answer questions and view your final score
